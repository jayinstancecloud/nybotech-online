# NYBOTECH Website

A modern, responsive website for NYBOTECH - a medical technology solutions provider, inspired by nybotech.asia and nybotech.store.

## Features

- **Homepage**: Hero section, features, and featured products showcase
- **Products Page**: Complete product catalog with category filtering
- **About Page**: Company information, mission, vision, and values
- **Contact Page**: Contact form and contact information
- **Shopping Cart**: Full e-commerce functionality with cart management
- **Responsive Design**: Mobile-friendly design that works on all devices

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Local Storage for cart persistence

## Features Breakdown

### Navigation
- Sticky navigation bar
- Mobile-responsive hamburger menu
- Shopping cart icon with item count
- Active page highlighting

### Products
- 12 medical equipment products across 4 categories:
  - Diagnostic (stethoscope, ultrasound, X-ray, MRI)
  - Monitoring (patient monitor, BP monitor, pulse oximeter)
  - Surgical (laser system, electrosurgical unit, surgical table)
  - Therapeutic (ventilator, infusion pump)
- Category filtering
- Add to cart functionality
- Product cards with descriptions and pricing

### Shopping Cart
- Add/remove items
- Quantity adjustment
- Order summary with subtotal, shipping, and total
- Checkout form with validation
- Cart persistence using localStorage

### Contact Form
- Contact information display
- Functional contact form
- Business hours information

## Getting Started

1. Open `index.html` in a web browser
2. Navigate through the pages using the navigation menu
3. Browse products and add items to cart
4. View cart and proceed to checkout
